# Skills Curated para StarTarot

Essas 12 skills foram selecionadas como mais relevantes pro stack do projeto:
- Next.js + React + Node.js + PostgreSQL + Supabase
- SaaS de consultas com vídeo
- Foco em deploy, performance e qualidade

## 📚 Skills incluídas

1. **api-patterns** — REST, GraphQL, rate limiting, autenticação
2. **nodejs-best-practices** — padrões Node.js, async/await, streams
3. **nextjs-react-expert** — otimizações Next.js (bundle, SSR, cache, re-renders)
4. **database-design** — schema, indexing, migrations, ORM
5. **deployment-procedures** — CI/CD, releases, rollbacks
6. **testing-patterns** — unit, integration, E2E
7. **architecture** — design patterns, context discovery, trade-offs
8. **systematic-debugging** — troubleshooting metodológico
9. **clean-code** — naming, functions, refactoring
10. **performance-profiling** — lighthouse, profiling, otimizações
11. **lint-and-validate** — linting, type checking, validação
12. **plan-writing** — planejamento estruturado de features

## ❌ Skills removidas

Game dev, mobile design, SEO, geolocation, red-team, RPA workflows, etc. — não se aplicam ao projeto.
